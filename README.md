SmartShop E-Commerce Chatbot

A simple and interactive E-Commerce Chatbot system built using Python and Web Technologies.
This project combines a CLI-based chatbot with a web-based shopping interface, providing a basic simulation of an online shopping platform with automated responses.

Features
 Chatbot for customer interaction
 Product browsing by categories
 Add, view, and clear cart
 Order tracking simulation
 Basic customer support responses
 Web interface + CLI support

🧠 How It Works

The system uses a rule-based chatbot approach, where user input is matched with predefined keywords.
